# 命令行

## init

运行配置初始化向导

```bash
yarn qiufen init
```

## start

启动 graphql 服务器

```bash
yarn qiufen start
```

## check

检查一些配置选项

```bash
yarn qiufen check
```

## help

显示命令帮助

```bash
yarn qiufen help
```
